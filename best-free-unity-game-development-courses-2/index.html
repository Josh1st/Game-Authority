Cost: R159 / m

Xbox Game Pass Console edition is for... You guessed it, Consoles! (the X-Bone ðŸ¦´). What's so special about the console version?

  • it gives you access to 228Â games
  • New games added all the time

So Is It Worth It?

I would say yes it is especially if you are on console. They are constantly adding more games and seem to be releasing some Microsoft games on Game Pass the same day as release. But not all games, for example, Quantum Break (released 2016) is a Microsoft exclusive on both Xbox and PC but it only included Xbox Game Pass Ultimate ðŸ˜’ BRUH, that's just silly to me.
